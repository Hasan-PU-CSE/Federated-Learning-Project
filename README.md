# Federated Learning on Drone Dataset 🚁

## 📌 Overview
This project implements Federated Learning on drone anomaly dataset 
with IID Clients.

## ⚙️ Technologies
- Python
- TensorFlow
- NumPy

## 🚀 Features
- 3 IID Clients
- 9 Non-IID Clients
- 100+ Training Rounds

## 📊 Results
- Accuracy: 99.98%
- Improved convergence speed
