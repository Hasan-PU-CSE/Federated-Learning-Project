# Federated Learning on Drone Dataset 🚁

## 📌 Overview
This project implements Federated Learning on drone anomaly dataset 
with IID Clients.

## ⚙️ Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- TensorFlow

## 🚀 Features
- 3 IID Clients
- 9 IID Clients
- 10+ Training Rounds

## 📊 Results
- Accuracy: 99.98%
- Improved convergence speed

## 📎 Dataset
[https://onlineacademiccommunity.uvic.ca/isot/2024/12/05/drone-datasets/]
